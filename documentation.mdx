<Note>
  **Agent First** documentation is compatible with the **Mintlify** platform out of the box. If you
  use another documentation platform, [let us know](mailto:brain@agentfirst.dev).
</Note>

## Mintlify integration

Follow the steps below to whitelabel our documentation within **Mintlify**.

<Steps>
  <Step title="Import project">
    Go to your Mintlify directory then import the whitelabel project by choosing a subdirectory name
    and URL path to nest the documentation under:

    ```shell
    git subtree add \
    --prefix='[subdirectory/pathname here]' \
    --squash \
    https://github.com/agentfirstdev/whitelabel \
    main
    ```

    <Accordion icon='code' title='View subdirectory/pathname example'>
      We chose `endpoints`, as in https://doc.agentfirst.dev/endpoints/search:

      ```shell
      git subtree add \
      --prefix=endpoints \
      --squash \
      https://github.com/agentfirstdev/whitelabel \
      main
      ```
    </Accordion>
  </Step>

  <Step title="Download dependencies">
    Switch to your new subdirectory then download the runtime dependencies:

    ```shell
    cd '[subdirectory/pathname here]'
    npm i
    ```
  </Step>

  <Step title="Brand documentation">
    Brand the documentation by calling the whitelabel script with your company name and API domain
    of choice:

    ```shell
    npm run whitelabel -- \
    --company='[company name here]'\
    --endpoint=[API domain here]
    ```

    <Accordion icon='code' title='View branding example'>
      Here's how we branded our documentation:

      ```shell
      npm run whitelabel -- \
      --company='Agent First' \
      --endpoint=api.agentfirst.dev
      ```
    </Accordion>
  </Step>

  <Step title="Preview changes">
    Preview your changes as usual:

    ```shell
    mint dev
    ```

    A local copy of your documentation will be hosted at http://localhost:3000/.
  </Step>

  <Step title="Publish changes">
    Publish your changes after reviewing them:

    ```shell
    git push
    ```

    If you've connected your repository to **Mintlify**, your documentation will be automatically
    deployed.
  </Step>

  <Step title="Incorporate updates">
    To incorporate documentation updates later, sync the whitelabel project again:

    ```shell
    git subtree pull \
    --prefix='[subdirectory/pathname here]' \
    --squash \
    https://github.com/agentfirstdev/whitelabel \
    main
    ```

    Then, repeat the directions from **step 2**.
  </Step>
</Steps>
