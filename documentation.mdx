## Platform support

The API documentation plugs directly into **Mintlify**. If you're using another documentation
platform, [let us know](mailto:brain@agentfirst.dev).

## Mintlify integration

Follow the steps below to whitelabel the documentation in **Mintlify**.

<Steps>
  <Step title="Import API documentation">
    Go to your Mintlify directory then import the API documentation by specifying a subdirectory
    name and URL path you want to use:

    ```shell
    git subtree add \
    --prefix='[Insert subdirectory/pathname here]' \
    --squash \
    https://github.com/agentfirstdev/whitelabel \
    main
    ```

    <Accordion icon='code' title='View subdirectory/pathname example'>
      We used `endpoints`, as in https://doc.agentfirst.dev/endpoints/search:

      ```shell
      git subtree add \
      --prefix=endpoints \
      --squash \
      https://github.com/agentfirstdev/whitelabel \
      main
      ```
    </Accordion>
  </Step>
</Steps>
